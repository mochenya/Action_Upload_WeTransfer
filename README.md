# 使用 Github Action 上传文件到 WeTransfer

## 简介 :

     WeTransfer 国外的一个基于云端的大文件传输工具

     凭借着WeTransfer，用户可以免登录上传2G以内的文件

## 如何使用

#### 1、点击本仓库右上角的 'Fork'

#### 2、然后会自动跳转到 Fork 的仓库

#### 3、依次点击 'Actions - Select workflow - Upload_WeTransfer - Run workflow'

#### 4、填写好你的参数 最后点击绿色的'Run workflow'

#### 5、耐心等待工作
